# Disease Prediction with Medical Recommendation - Learning Demonstration

This repository is a personal machine-learning demonstration built for learning. It explores symptom-based classification and recommendation-style output using a prepared dataset.

It is not a diagnostic system, medical device, clinical decision-support product, or source of treatment advice. Accuracy on a prepared dataset does not establish clinical validity, safety, generalization, or suitability for patient use.

## Appropriate use

- Inspect the data-processing and model-training workflow.
- Reproduce results in a local learning environment.
- Study model limitations, leakage risk, class imbalance, and evaluation design.

## Inappropriate use

- Do not use outputs to diagnose, treat, triage, or delay professional care.
- Do not enter personal health information.
- Do not describe the project as clinically validated or deployed.

## Future evidence needed

A stronger learning version would document dataset provenance and licensing, train/test separation, per-class metrics, calibration, failure analysis, subgroup limitations, reproducible environment setup, and a clear non-clinical interface.
